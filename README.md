# ⏱️ DeepTimer

A simple stopwatch app built with vanilla HTML, CSS, and JavaScript.
Tracks hours, minutes, seconds, and milliseconds. No frameworks, no libraries — just plain JS.

## Features

- Start, pause, and reset
- Displays hours, minutes, seconds, and milliseconds
- Lap time recording
- Built with vanilla JS using `setInterval` and `clearInterval`
- Works entirely in the browser — no install needed

## Run it

Clone the repo and open `index.html` in your browser. That's it.

```bash
git clone https://github.com/bekwizgobisa321-lgtm/deeptimer.git
cd DeepTimer
open index.html
```


## Tech

- HTML
- CSS
- Vanilla JavaScript

## Why I built this

A hands-on exercise to learn DOM manipulation and how `setInterval` works in JavaScript. No AI, no copy-paste — written from scratch.

Good starting point if you're also learning the basics of JavaScript timers and event listeners.
